Plain JS Subtyping App

An example of a front-end web app with subtyping/inheritance in class hierarchies, built with plain JavaScript

  • Books
  • Authors
  • Employees
  • Persons

This app supports the following operations:

  • Manage book data
  • Manage author data
  • Manage employee data
  • Manage person data

For playing with the app, you may first want to and then view it (e.g. by choosing "Manage books" and then "List all books").

This example app takes care of two class hierarchies:

  • The simple segmentation of Book into TextBook and Biography.
  • The segmentation of Person into Author and Employee such that Employee has a further segment, Manager.
